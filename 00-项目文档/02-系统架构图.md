## 二、单产品处理流程（详细）

```
步骤 1：读取产品数据
  从 Excel/CSV 中读取一行 = 一个产品的所有信息
  例如：{
    "product_name": "Garden Fork",
    "material": "Carbon Steel + Wood Handle",
    "weight": "0.8 kg",
    "size": "4 tines, 30cm",
    ...
  }

步骤 2：组装 Prompt
  把产品数据填入 Prompt 模板
  模板 + 数据 = 完整的 AI 指令

步骤 3：调用 AI
  把完整的 Prompt 发给 Qwen3.6-plus
  等待 AI 返回结果

步骤 4：解析结果
  AI 返回的内容可能是 JSON、文本等
  提取你需要的部分

步骤 5：保存结果
  把结果保存到对应的文件
  文件名包含产品名，方便查找

步骤 6：下一个产品
  重复步骤 1-5，直到所有产品处理完
```

## 三、AI 调用逻辑（用外卖比喻）

想象你开了一家餐厅：

```
1. 接单（读取产品数据）
   → 客人点了一份宫保鸡丁
   → 你从菜单上查到这个菜的原料

2. 备料（组装 Prompt）
   → 把原料准备好：鸡肉、花生、辣椒
   → 把产品信息准备好：名称、材质、规格
   → 按照菜谱（Prompt 模板）排列好

3. 炒菜（调用 AI）
   → 把原料放进锅里炒
   → 把 Prompt 发给 Qwen
   → 等待结果（AI 帮你"炒"出文案）

4. 装盘（解析结果）
   → 把炒好的菜装到盘子里
   → 把 AI 返回的内容整理好格式

5. 上菜（保存输出）
   → 把菜端给客人
   → 把文案保存到文件

6. 下一单（循环）
   → 重复以上步骤
```

## 四、输入输出关系表

| 输入             | 处理（AI）          | 输出           | 格式      |
| -------------- | --------------- | ------------ | ------- |
| 产品参数（1行 Excel） | SEO 标题 Prompt   | 3-5 个 SEO 标题 | CSV     |
| 产品参数 + 关键词库    | 卖点描述 Prompt     | 5 个产品卖点      | CSV     |
| 产品参数 + 工厂信息    | 阿里详情 Prompt     | 详情页 HTML/MD  | HTML/MD |
| 产品参数 + 客户类型    | WhatsApp Prompt | 开发话术 3 版     | TXT     |
| RFQ 内容 + 产品参数  | RFQ 回复 Prompt   | RFQ 回复模板     | TXT     |
| 产品参数 + 目标国家    | 市场版 Prompt      | 市场定制文案       | TXT     |
| 所有产品信息         | 知识库生成 Prompt    | 产品知识卡片       | JSON    |

## 六、知识库在工作流中的位置

```
知识库不是"一次性"的，而是持续积累的：

┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  产品知识库   │     │  市场知识库   │     │  客户知识库   │
│              │     │              │     │              │
│ · 产品参数    │     │ · 关税信息    │     │ · 客户类型    │
│ · 使用场景    │     │ · 市场偏好    │     │ · 关注点      │
│ · 竞争优势    │     │ · 认证要求    │     │ · 沟通习惯    │
│ · 常见问题    │     │ · 节日/旺季   │     │ · 采购周期    │
└──────┬───────┘     └──────┬───────┘     └──────┬───────┘
       │                    │                    │
       └────────────────────┼────────────────────┘
                            ▼
                    注入到 Prompt 中
                    作为 AI 的背景知识

结果：AI 生成的内容更专业、更精准
```

## 七、关键概念速查

| 术语     | 简单解释             | 外卖比喻     |

| 循环     | 重复做同样的事          | 一份一份做菜   |
| API    | 调用 AI 的方式        | 打电话给厨师   |
