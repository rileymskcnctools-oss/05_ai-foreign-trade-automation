# AI 外贸产品运营自动化系统 — 项目总览

> 总教练模式 | 3 周从 0 到 1
> 产品方向：手动农具 / Garden Tools / Farm Tools

---

## 一、项目目标

```
输入：Excel 产品参数表（几十个产品，每行一个产品）
      ↓
AI 自动处理（通过 Hermes + Qwen3.6-plus）
      ↓
输出：完整的外贸营销内容包
  ├── SEO 标题（每个产品 3-5 个变体）
  ├── 五点卖点 / 产品优势
  ├── 阿里国际站详情页文案（HTML 结构）
  ├── WhatsApp 开发话术
  ├── RFQ 快速回复模板
  ├── 国家市场版文案（按目标市场定制）
  ├── 产品知识库（结构化 JSON）
  └── （后期）PDF 产品资料
```

---

## 二、3 周学习路线总览

### 第 1 周：基础搭建（Day 1-7）

- 理解 AI 工作流的基本概念
- 设计并创建 Excel 产品数据库
- 学习 Prompt 工程基础
- 建立第一批 Prompt 模板
- 用 Hermes 手动跑通一个产品的完整生成流程

### 第 2 周：系统化（Day 8-14）

- 建立完整的 Prompt 模板库
- 学习 CSV / JSON 数据结构
- 用 Hermes 编写批量处理工作流
- 建立知识库系统
- 测试并优化生成质量

### 第 3 周：自动化 & 交付（Day 15-21）

- 完善批量处理脚本
- 添加国家市场适配
- 建立质量检查流程
- 整理最终交付文档
- 独立跑通一个完整项目（10+ 产品）

### 每日学习节奏

```
每天 2 小时分配：
  30 分钟 → 学习概念（我教你）
  60 分钟 → 动手做（我带着你做）
  30 分钟 → 总结 & 记录
```

---

## 三、最终交付物

完成 3 周后，你将拥有：

1. 一个完整的 Excel 产品数据库模板
2. 一套可直接复制使用的 Prompt 模板库
3. 一个 Hermes 自动化工作流（读取 Excel → AI 处理 → 输出内容）
4. 四个知识库（产品、市场、客户、关键词）
5. 一份项目操作手册
6. 至少 10 个产品的完整营销内容输出
