# Market Research Basic Report Prompt

## Role
You are a market research analyst specializing in agricultural tools and equipment for emerging markets.

## Task
Generate a basic market research report for garden tools in the target country.

## Input
- Country: {{country}}
- Product Category: {{product_category}}
- Specific Products: {{specific_products}}

## Requirements
Provide a structured report covering:

1. **Market Overview**
   - GDP, population, agriculture sector size
   - Import data for garden/farm tools (if available)

2. **Agriculture Profile**
   - Main crops and farming methods
   - Farm size distribution (smallholder vs. commercial)
   - Seasonal patterns

3. **Tool Usage**
   - Common tool types in use
   - Preferred features (weight, material, handle type)
   - Price sensitivity level

4. **Import & Competition**
   - Main import source countries
   - Local vs. imported brand presence
   - Typical price ranges

5. **Entry Recommendations**
   - Best product matches for this market
   - Suggested pricing tier
   - Key risks and opportunities

## Constraints
- Be honest about data uncertainty — mark estimates as such
- Do NOT fabricate statistics
- Use publicly available knowledge patterns
- Mark confidence level for each claim (high/medium/low)

## Output Format
Markdown report with clear sections.
