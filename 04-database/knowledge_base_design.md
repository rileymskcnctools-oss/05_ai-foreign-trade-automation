# 知识库系统设计

> 知识库 = 你的外贸经验数字化
> 不是"一次性"的东西，而是持续积累的资产

---

## 一、为什么需要知识库？

想象一下：

- 没有知识库：每次生成内容，AI 都不知道你卖什么、目标市场是什么
- 有了知识库：AI 能调用你的经验，生成更专业、更精准的内容

**知识库的本质**：把你的经验变成 AI 能读懂的"参考资料"

---

## 二、四大知识库

### 1. 产品知识库

**内容**：每个产品的完整信息卡片
**用途**：AI 生成任何产品内容时参考

**结构设计**：

```markdown
## GF-001 | Garden Fork | 园林叉

### 基本信息
- 英文名：Garden Fork
- 中文名：园林叉
- 类别：Digging Tools > Forks
- 编号：GF-001

### 技术参数
- 材质：Carbon Steel (高碳钢)
- 手柄：Ash Wood (白蜡木)
- 齿数：4
- 长度：30cm
- 重量：0.8kg
- 硬度：HRC 45-50
- 表面处理：Polished + Lacquered

### 商业信息
- MOQ：500 pcs
- 包装：Color Box
- 每箱：20 pcs
- 外箱：62x32x35 cm
- 毛重：17.5 kg/箱
- 交货期：25 天
- 认证：CE, GS

### 卖点提炼
1. HRC 45-50 热处理，不变形
2. 白蜡木手柄，人体工学
3. 双重防锈处理
4. 多场景适用

### 竞品信息
- Fiskars X3: 更贵，品牌溢价高
- Spear & Jackson: 英国品牌，质量对标
- 我们的优势：同等质量，价格低 30%

### 客户反馈
- "手柄很舒服" — 德国客户
- "用了一年没变形" — 尼日利亚客户
- "性价比很高" — 巴西客户
```

**积累方法**：

- 每开发一个新产品，就建一张卡片
- 每次收到客户反馈，更新到卡片中
- 每次发现竞品新动态，更新竞品信息

### 2. 市场知识库

**内容**：每个目标市场的详细信息
**用途**：生成国家市场版文案时参考

**结构设计**：

```markdown
## 🇳🇬 Nigeria | 尼日利亚

### 市场概况
- 人口：2.2 亿
- GDP 排名：非洲第一
- 农业占比：约 25% GDP
- 主要作物：木薯、玉米、水稻

### 农具市场需求
- 主要用途：小规模农场、家庭种植
- 偏好类型：重型、耐用、多功能
- 价格区间：$1-5/件（零售）
- 年进口量：估计 XXX 万件

### 渠道分布
- 主要批发市场：Lagos (Idumota), Kano (Kantin Kwari)
- 主要进口商：XXX 家
- 线上渠道：Jumia, Konga

### 认证要求
- 无强制认证
- SONCAP 推荐（部分产品）

### 物流信息
- 主要港口：Lagos (Apapa, Tincan)
- 中国到 Lagos 船期：30-35 天
- 关税：农具类约 X%

### 竞品信息
- 主要来源：中国、印度
- 印度竞品：价格低 20%，质量差
- 中国竞品：同质化严重

### 搜索关键词
- garden fork, farm fork, digging fork
- heavy duty fork, agricultural tools
- 本地语言词（Hausa, Yoruba）

### 注意事项
- 雨季影响物流，提前备货
- 偏好大容量订单（价格敏感）
- 关系型商业文化，信任重要
```

**积累方法**：

- 每开拓一个新市场，建一个市场卡片
- 每次和客户聊到市场信息，记录下来
- 关注行业新闻，更新市场动态

### 3. 客户知识库

**内容**：客户画像和沟通策略
**用途**：生成客户开发话术时参考

**结构设计**：

```markdown
## 客户类型：批发商 (Wholesaler)

### 典型特征
- 采购量：1000-10000+ pcs/次
- 关注点：价格、交期、质量稳定性
- 决策周期：2-4 周
- 复购率：高（如果质量好）

### 沟通要点
- 重点谈：价格阶梯、MOQ、交期
- 展示：产能、质检流程、认证
- 避免：太长的产品介绍

### 话术策略
- 开场：直接报价 + 核心优势
- 跟进：样品 + 质检报告
- 逼单：价格有效期、产能紧张

### 常见异议
- "价格太高" → 展示质量对比、长期成本
- "交期太长" → 展示生产排期、备选方案
- "之前有质量问题" → 展示质检流程、质保承诺

---

## 客户类型：零售商 (Retailer / Garden Center)

### 典型特征
- 采购量：200-2000 pcs/次
- 关注点：产品外观、消费者卖点、利润空间
- 决策周期：1-3 周
- 复购率：中（看销售情况）

### 沟通要点
- 重点谈：包装设计、消费者卖点、利润
- 展示：产品图片、包装效果、零售价建议
- 避免：太多技术参数

### 话术策略
- 开场：产品亮点 + 利润空间
- 跟进：包装方案、营销素材
- 逼单：季节性促销、限量优惠
```

**积累方法**：

- 每次和客户沟通后，记录关键信息
- 总结不同类型客户的共同特点
- 记录成功/失败的沟通案例

### 4. 关键词库

**内容**：各品类、各市场的搜索关键词
**用途**：SEO 标题和产品描述的关键词来源

**结构设计**：

```markdown
## Garden Fork | 园林叉 — 关键词库

### 核心大词（搜索量高，竞争大）
- garden fork
- digging fork
- garden tools

### 精准词（搜索量中，转化率高）
- 4 tine garden fork
- steel garden fork
- heavy duty garden fork
- garden fork with wooden handle

### 长尾词（搜索量低，转化率极高）
- carbon steel garden fork for heavy soil
- garden fork with ergonomic handle
- professional grade garden fork

### 市场特定词
- Nigeria: farm fork, agricultural fork
- Germany: Gartengabel, Bodengruber
- Brazil: garfo de jardim, enxada de jardinagem
- Kenya: farming fork, digger fork

### 竞品品牌词
- fiskars garden fork alternative
- spear and jackson fork equivalent

### 场景词
- compost fork
- lawn aeration fork
- soil turning tool
- manure fork
```

**积累方法**：

- 用阿里国际站后台的关键词工具
- 用 Google Keyword Planner
- 记录客户在 RFQ 中使用的词
- 定期更新（每月的趋势会变）

## 三、知识库的使用方式

### 在 Prompt 中注入知识库

```
【产品知识库】
（从产品知识库中复制对应产品卡片）

【市场知识库】
（从市场知识库中复制目标市场卡片）

【请根据以上信息，生成...】
```

### 知识库更新流程

```
每次完成一个任务后：
  1. 有没有新的产品信息？ → 更新产品知识库
  2. 有没有新的市场洞察？ → 更新市场知识库
  3. 有没有新的客户反馈？ → 更新客户知识库
  4. 有没有新的关键词？ → 更新关键词库

建议：每周花 15 分钟整理知识库
```
