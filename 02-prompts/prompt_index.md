# Prompt 模板库总览

> 所有 Prompt 模板都可以直接复制使用
> 用 `{大括号}` 标记的部分是需要替换的变量

---

## 模板列表

| 文件名                 | 用途               | 输入            | 输出      |
| ------------------- | ---------------- | ------------- | ------- |
| SEO标题Prompt.md      | 生成阿里国际站 SEO 标题   | 产品参数          | 5 个标题变体 |
| 卖点描述Prompt.md       | 生成五点产品卖点         | 产品参数          | 5 条卖点   |
| 阿里详情Prompt.md       | 生成阿里详情页文案        | 产品参数 + 工厂信息   | 完整详情页   |
| WhatsApp话术Prompt.md | 生成 WhatsApp 开发话术 | 产品参数 + 客户类型   | 3 版话术   |
| RFQ回复Prompt.md      | 生成 RFQ 快速回复      | RFQ 内容 + 产品参数 | RFQ 回复  |
| 国家市场Prompt.md       | 生成国家市场版文案        | 产品参数 + 目标国家   | 市场定制文案  |

---

## 使用方法

1. 打开对应的 Prompt 模板
2. 把 `{大括号}` 里的内容替换成实际的产品信息
3. 把完整的 Prompt 发给 Qwen3.6-plus
4. 复制 AI 返回的结果

## 批量使用

后期用 Hermes 自动化时，Hermes 会自动：

- 读取 Excel 中的一行数据
- 把数据填入 `{大括号}`
- 发给 AI
- 保存结果
