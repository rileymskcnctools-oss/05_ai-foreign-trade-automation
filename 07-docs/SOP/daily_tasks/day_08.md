## 学习任务（30分钟）

### Power BI数据导入流程

1. 打开Power BI Desktop
2. 点击"获取数据" → "文本/CSV"
3. 选择文件 → 编辑（进入Power Query）
4. 检查数据类型 → 应用并关闭

### 数据类型检查

| 字段类型 | Power BI中的正确类型 |
|---------|---------------------|
| 产品ID/ASIN | 文本 |
| 价格 | 十进制数 |
| 评分 | 十进制数 |
| 评论数 | 整数 |
| 是否FBA | 是/否 |
