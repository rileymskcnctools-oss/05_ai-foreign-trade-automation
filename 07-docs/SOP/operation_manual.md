# 操作手册

> AI 外贸产品运营自动化系统
> 版本：1.0
> 更新日期：2026-05-27

---

## 一、系统概览

本系统用于：

- 输入：Excel 产品参数表
- 输出：完整的外贸营销内容包（SEO 标题、卖点、阿里详情、WhatsApp 话术、RFQ 回复、市场版文案）

使用工具：

- Hermes（AI Agent）
- Qwen3.6-plus（大语言模型）
- Excel（产品数据库）

---

## 二、快速开始

### 第一步：准备产品数据

1. 打开 Excel，按 `01-产品数据库模板/字段说明.md` 中的字段设计创建表格
2. 录入你的产品信息（至少 1 个产品）
3. 保存为 `产品参数表.xlsx`

### 第二步：选择要生成的内容类型

| 内容类型        | Prompt 模板位置                         | 输出格式     |
| ----------- | ----------------------------------- | -------- |
| SEO 标题      | 02-Prompt模板库/01-SEO标题Prompt.md      | JSON     |
| 卖点描述        | 02-Prompt模板库/02-卖点描述Prompt.md       | JSON     |
| 阿里详情        | 02-Prompt模板库/06-阿里详情Prompt.md       | Markdown |
| WhatsApp 话术 | 02-Prompt模板库/03-WhatsApp话术Prompt.md | JSON     |
| RFQ 回复      | 02-Prompt模板库/04-RFQ回复Prompt.md      | JSON     |
| 国家市场版       | 02-Prompt模板库/05-国家市场Prompt.md       | JSON     |

### 第三步：手动生成（新手推荐）

1. 打开对应的 Prompt 模板
2. 把 `{大括号}` 替换成实际产品信息
3. 把完整的 Prompt 发给 Qwen3.6-plus
4. 复制 AI 返回的结果，保存到 `05-输出结果/` 文件夹

### 第四步：批量生成（熟练后）

让 Hermes 自动批量处理：

1. 告诉 Hermes 产品表的位置
2. 告诉 Hermes 要生成哪些内容
3. Hermes 自动处理所有产品
4. 完成后检查结果

## 三、详细操作步骤

### A. 手动生成 SEO 标题

1. 打开 `02-Prompt模板库/01-SEO标题Prompt.md`
2. 复制全部 Prompt 内容
3. 替换所有 `{大括号}` 为实际产品数据：
   - `${product_id}` → GF-001
   - `${product_name_en}` → Garden Fork
   - `${category}` → Digging Tools
   - ...（所有带大括号的都替换）
4. 把替换后的完整 Prompt 发给 Qwen
5. 复制 AI 返回的 JSON 结果
6. 保存到 `05-输出结果/GF-001/seo_titles.json`

### B. 手动生成卖点描述

步骤同上，使用 `02-卖点描述Prompt.md`。

### C. 手动生成 WhatsApp 话术

步骤同上，使用 `03-WhatsApp话术Prompt.md`。
需要额外指定：

- `${customer_type}` → wholesaler / retailer / distributor
- `${target_market}` → Nigeria / Germany / Brazil 等

### D. 手动生成 RFQ 回复

步骤同上，使用 `04-RFQ回复Prompt.md`。
需要额外提供：

- `${rfq_content}` → 复制买家的 RFQ 原文
- `${price_range}` → 你的报价范围

### E. 手动生成国家市场版

步骤同上，使用 `05-国家市场Prompt.md`。
需要额外提供：

- `${target_country}` → 目标国家
- `${market_info}` → 从 `04-知识库/市场知识库.md` 复制对应国家的信息

## 四、批量处理操作

### 让 Hermes 批量处理 10 个产品

对 Hermes 说：

```
请帮我批量处理以下任务：

1. 读取桌面上的产品参数表（01-产品数据库模板/产品参数表.xlsx）
2. 对每个产品，执行以下操作：
   a. 读取该产品的所有信息
   b. 用 SEO 标题 Prompt 模板（02-Prompt模板库/01-SEO标题Prompt.md）生成 5 个 SEO 标题
   c. 用卖点描述 Prompt 模板（02-Prompt模板库/02-卖点描述Prompt.md）生成 5 个卖点
   d. 把结果分别保存到 05-输出结果/[product_id]/ 目录下
3. 全部完成后，生成一个汇总报告，列出所有产品的处理状态
```

## 五、文件命名规则

### 输入文件

- 产品参数表：`产品参数表.xlsx` 或 `products.csv`

### 输出文件

- 单个产品：`05-输出结果/[product_id]/[内容类型].[格式]`
  - 示例：`05-输出结果/GF-001/seo_titles.json`
- 汇总文件：`05-输出结果/汇总/all_[内容类型].csv`
  - 示例：`05-输出结果/汇总/all_seo_titles.csv`

## 六、质量检查

每次生成内容后，检查以下项目：

### SEO 标题检查

- [ ] 每个标题不超过 128 字符
- [ ] 核心关键词在标题前部
- [ ] 5 个标题有不同侧重点
- [ ] 没有无意义词（Hot Sale, Best Price）

### 卖点描述检查

- [ ] 5 个卖点覆盖不同维度
- [ ] 有 Feature → Benefit 转换
- [ ] 有具体数据支撑
- [ ] 面向 B2B 买家

### WhatsApp 话术检查

- [ ] 不超过 500 字符
- [ ] 语气自然，不像群发
- [ ] 有具体的产品数据
- [ ] 结尾有低压力引导

### 国家市场版检查

- [ ] 体现了目标市场特点
- [ ] 使用了当地搜索词
- [ ] 提到了当地竞品或渠道
- [ ] FAQ 是当地买家真正会问的

## 七、常见问题

### Q1: AI 生成的内容不满意怎么办？

A: 修改 Prompt 后重新生成。常见问题：

- 内容太泛 → 在 Prompt 中加更多产品细节
- 内容太长 → 在 Prompt 中限制字数
- 语气不对 → 在 Prompt 中描述期望的语气

### Q2: 如何处理大量产品（50+）？

A: 分批处理，每批 10 个。全部处理完后合并汇总文件。

### Q3: 如何更新知识库？

A: 每次有新的信息时，直接编辑对应的知识库文件。建议每周整理一次。

### Q4: AI 返回的 JSON 格式不对怎么办？

A: 在 Prompt 末尾加一句：

```
请严格使用以上 JSON 格式输出，不要添加其他内容。
```
