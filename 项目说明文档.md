## 一、项目是什么

这是一个**AI驱动的外贸自动化内容生成系统**，利用AI（Hermes + Qwen）从产品数据库自动生成外贸所需的各类营销内容。

核心流程：**产品目录PDF → 提取产品参数到Excel → AI生成SEO标题/详情/RFQ回复/WhatsApp话术 → 输出可直接用于跨境平台的上架内容。**

**项目关键数据：**
- 已提取产品：211个（来自天津Toolsmart 2026产品目录PDF，59页）
- 产品品类：除草工具(81)、挖掘工具(74)、切割工具(60)
- 目标市场：非洲、欧洲、北美、南美
- 已生成产品内容：GF-001、GR-001、GS-001、HT-001、SH-001（5个产品）

**技术栈：** Python(pandas/PyMuPDF) + AI提示词工程 + Hermes自动化 + Excel数据库

```

│
├── seo/
│   └── seo_title_prompt.md            (3.8KB) ← ★ SEO标题生成提示词
│       用途：为每个产品生成3个版本的SEO标题
│       输出：适用于阿里国际站、独立站、Google SEO
│
├── detail_page/
│   ├── selling_points_prompt.md       (5.7KB) ← ★ 产品卖点生成提示词
│   │   用途：生成产品核心卖点、优势描述
│   │   输出：5-8条卖点，按重要性排序
│   │
│   └── alibaba_detail_prompt.md       (3.5KB) ← 阿里国际站详情页提示词
│       用途：生成完整的阿里国际站产品详情页内容
│       输出：HTML/Markdown格式的详情页
│
├── rfq/
│   └── rfq_reply_prompt.md            (5.7KB) ← ★ RFQ（询价）回复提示词
│       用途：自动生成RFQ回复模板
│       输出：专业、有说服力的询价回复
│
├── whatsapp/
│   └── whatsapp_script_prompt.md      (4.7KB) ← WhatsApp营销话术提示词
│       用途：生成WhatsApp沟通话术
│       输出：首次联系、跟进、报价等场景话术
```

**使用顺序：**
1. 先看 `prompt_index.md` 了解整体
2. 然后按业务需求看对应的提示词：
   - 要做SEO优化 → 看 `seo/seo_title_prompt.md`
   - 要写产品详情 → 看 `detail_page/` 下的提示词
   - 要回复客户询价 → 看 `rfq/rfq_reply_prompt.md`
   - 要做WhatsApp营销 → 看 `whatsapp/whatsapp_script_prompt.md`
   - 要针对不同市场定制内容 → 看 `batch_workflow/market_localization_prompt.md`

## 五、你应该怎么使用这个项目

### 场景1：生成新产品内容

1. 确保产品在 `04-database/excel/product_database_filled.xlsx` 中
2. 打开 `02-prompts/` 下对应的提示词模板
3. 用Hermes调用AI生成内容
4. 结果保存到 `05-output/export_files/[产品编码]/`

### 场景2：继续学习计划

1. 打开 `07-docs/SOP/daily_tasks/`
2. 找到你上次停下的 `day_XX.md`
3. 按任务清单执行

### 场景3：修改提示词

1. 打开 `02-prompts/` 下对应的提示词
2. 修改后保存
3. 重新运行工作流生成新内容

### 场景4：添加新产品到数据库

1. 打开 `04-database/excel/product_database_filled.xlsx`
2. 在最后一行后新增
3. 按 `00-config/field_mapping.md` 的规范填写
4. 保存后就可以用AI生成内容了
